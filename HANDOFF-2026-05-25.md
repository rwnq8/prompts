**Approach A (Recommended): Zero-dependency translator**
Rewrite `translate_text()` to use `urllib.request` + `json` (both stdlib) to call Google Translate directly. No external packages needed. Example endpoint:

```python
import urllib.request
import urllib.parse
import json

def translate_text(text, dest_lang, source_lang='en'):
    """Translate using Google Translate API (no external deps)."""
    url = "https://translate.googleapis.com/translate_a/single"
    params = {
        "client": "gtx",
        "sl": source_lang,
        "tl": dest_lang,
        "dt": "t",
        "q": text
    }
    full_url = url + "?" + urllib.parse.urlencode(params)
    req = urllib.request.Request(full_url, headers={"User-Agent": "Mozilla/5.0"})
    with urllib.request.urlopen(req) as resp:
        data = json.loads(resp.read().decode())
    # Google returns [[["translated text", "original", ...]], ...]
    result = ""
    for segment in data[0]:
        if segment[0]:
            result += segment[0]
    return result
```
